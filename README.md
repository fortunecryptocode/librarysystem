# Library System 
A simple library management system built with Python. 
This project allows users to manage books, track borrowers, and handle basic library operations in an organized way.
## Features - Add, update, and remove books - Search for books by title or author - Track borrowed and returned books - Manage borrower information - Simple, user-friendly interface
